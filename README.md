# AI Nutrition & Meal Intelligence System

An AI-powered nutrition analysis system that uses Computer Vision and Natural Language Processing (NLP) to recognize food, calculate nutritional values, and provide personalized dietary recommendations.

## Features
- Food image recognition using CNN
- NLP-based meal text analysis
- Nutrition tracking dashboard
- Personalized recommendation engine
- Bangladeshi food dataset support

## Tech Stack
- Python
- FastAPI
- TensorFlow / PyTorch
- spaCy
- MySQL
- OpenCV
- Streamlit

## Project Structure
backend/ - FastAPI backend
frontend/ - UI frontend
dataset/ - Food datasets
trained_models/ - Saved AI models
